# PICK 보안 강화 + 자동 관리자 제거 버전

변경 내용:
- 처음 접속 시 로그인 화면 우선
- 회원가입만으로는 아무도 관리자가 되지 않음
- 첫 번째 가입자 자동 관리자 기능 제거
- `PICK_ADMIN_USERNAME` 환경변수와 같은 아이디로 가입한 경우에만 관리자
- 비밀번호 규칙 강화: 영어+숫자 포함 8자 이상
- 한글 아이디/비밀번호 차단 유지
- 로그인 실패 5회 제한
- 세션 보안 설정
- 보안 헤더 추가
- 관리자는 자기 자신의 관리자 권한 해제 불가
- 관리자는 자기 자신 삭제 불가
- 관리자 계정 삭제 불가

Render 환경변수 예:

```text
PICK_SECRET_KEY=긴랜덤문자열
PICK_ADMIN_USERNAME=minseok
PICK_COOKIE_SECURE=1
```
