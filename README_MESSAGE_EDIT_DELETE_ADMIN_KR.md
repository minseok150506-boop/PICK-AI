# PICK 메시지 삭제/수정 + 관리자 전체 삭제 버전

변경 내용:
- 관리자 활동 로그 메뉴 제거
- 메시지 삭제 기능 추가
- 사용자 메시지 수정 기능 추가
- 관리자만 전체 채팅/메시지 삭제 가능
- 관리자 페이지에서 “관리자님” 대신 로그인한 아이디 표시
- 로그아웃 다음에 “관리자 메뉴 닫기” 추가
- 대시보드 카드 영역 제거
- 사용자별 데이터 분리 유지

동작 규칙:
- 사용자는 자기 채팅방 메시지만 삭제할 수 있습니다.
- 사용자는 자기 사용자 메시지만 수정할 수 있습니다.
- AI 답변은 수정할 수 없습니다.
- 관리자는 전체 채팅/메시지를 삭제할 수 있습니다.
- 다른 사용자의 메시지에 직접 API 요청해도 403으로 차단됩니다.

적용 방법:
1. ZIP 압축 풀기
2. 내부 파일 전체를 GitHub 루트에 업로드
3. Commit changes
4. Render → Manual Deploy → Deploy latest commit
