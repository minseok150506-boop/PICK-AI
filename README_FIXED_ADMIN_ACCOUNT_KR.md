# PICK 고정 관리자 계정 버전

관리자 조건:
- 아이디: minseok
- 비밀번호: kms0506a!
- 위 두 조건이 모두 맞을 때만 관리자입니다.

중요:
- 첫 번째 가입자 자동 관리자 기능은 제거되었습니다.
- 관리자 코드 방식도 제거되었습니다.
- 다른 컴퓨터에서 가입해도 관리자 조건이 맞지 않으면 일반 유저입니다.
- 기존에 minseok 계정이 일반 유저로 만들어져 있어도, 정확한 비밀번호로 로그인하면 관리자 권한이 보정됩니다.

보안 주의:
이 방식은 개발/테스트용으로는 편하지만, 실제 서비스에서는 관리자 비밀번호를 코드에 고정하는 방식은 권장하지 않습니다.
실제 운영에서는 환경변수나 별도 관리자 승인 기능을 쓰는 것이 더 안전합니다.

적용 방법:
1. ZIP 압축 풀기
2. 내부 파일 전체를 GitHub 루트에 업로드
3. Commit changes
4. Render → Manual Deploy → Deploy latest commit
