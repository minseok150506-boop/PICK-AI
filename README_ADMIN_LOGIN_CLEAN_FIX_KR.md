# PICK 관리자 로그인/자동완성 꼬임 최종 수정

수정 내용:
- 로그인 화면에서 저장된 관리자 계정 자동완성 방지
- login_id/login_password 필드로 변경
- 로그인 GET 시 꼬인 세션 정리
- 관리자는 오직 minseok / kms0506a! 조합만 가능
- 일반 유저는 DB is_admin 값이 1이어도 관리자 세션을 받지 않음
- /login HEAD 요청 안전 처리
- 관리자 권한 정리 버튼 유지

적용 후:
1. ZIP 내부 파일 전체 업로드
2. Render 재배포
3. 브라우저에서 로그아웃 또는 /login 접속
4. 저장된 자동완성이 뜨면 브라우저 저장 비밀번호를 삭제
5. minseok / kms0506a! 로 로그인
6. /admin에서 관리자 권한 정리 클릭
