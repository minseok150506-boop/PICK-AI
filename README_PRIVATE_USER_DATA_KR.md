# PICK 사용자별 데이터 격리 버전

수정 내용:
- 각 사용자는 자기 채팅 목록만 볼 수 있습니다.
- 각 사용자는 자기 채팅 메시지만 볼 수 있습니다.
- 다른 사용자의 chat_id로 직접 접근해도 403 차단됩니다.
- 메시지 전송도 자기 채팅방에서만 가능합니다.
- 관리자 페이지는 사용자 수/메시지 수 같은 통계만 보고, 일반 사용자의 대화 내용은 기본적으로 보지 않습니다.
- DB 인덱스 추가: chats.user_id, messages.chat_id

적용 방법:
1. ZIP 압축 풀기
2. 내부 파일 전체를 GitHub 루트에 업로드
3. Commit changes
4. Render → Manual Deploy → Deploy latest commit

중요:
이미 다른 유저가 본 데이터가 브라우저에 남아 있으면 로그아웃 후 다시 로그인하거나 브라우저 캐시를 새로고침하세요.
