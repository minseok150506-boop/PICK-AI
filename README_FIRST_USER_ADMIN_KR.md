# PICK 첫 번째 가입자만 관리자 버전

관리자 규칙:
- 첫 번째 가입자만 관리자입니다.
- 두 번째 가입자부터는 모두 일반 유저입니다.
- 관리자는 자기 자신의 관리자 권한을 해제할 수 없습니다.
- 관리자는 자기 자신을 삭제할 수 없습니다.
- 관리자 계정은 삭제할 수 없습니다.

보안 유지:
- 처음 접속 시 로그인 화면 우선
- 한글 아이디/비밀번호 차단
- 비밀번호: 영어+숫자 포함 8자 이상
- 로그인 실패 제한
- 세션 보안 설정
- 보안 헤더 추가

주의:
이미 Render에 기존 DB가 있으면 첫 번째 유저 판정은 기존 DB 기준입니다.
처음부터 다시 시작하려면 기존 DB를 초기화해야 합니다.

Render:
Build Command:
pip install -r requirements.txt

Start Command:
gunicorn app:app
